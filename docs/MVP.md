# 家庭记账系统 MVP

## MVP 原则
- Less is More
- Mobile First
- Fast Input

## 功能范围
### 包含
- 登录
- 默认家庭
- 单成员
- 账户(Account)
- 内置分类(Category)
- 收入/支出(Transaction)
- Dashboard
- 流水
- 编辑/删除

### 不包含
- 转账
- 预算
- AI
- OCR
- 导入导出
- 投资
- 多币种

## 页面
- /login
- /dashboard
- /transaction/new
- /transactions
- /settings

## 完成标准
- 能登录
- 能创建账户
- 能新增收入/支出
- 能编辑删除
- 能查看流水
- 能查看首页统计
- Docker 一键启动
