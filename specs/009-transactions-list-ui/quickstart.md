# Quickstart: 009-transactions-list-ui 验证指南

**Date**: 2026-07-07

## 前置条件

- 001-008 全部就绪,`pnpm dev` 运行
- 已注册用户 + 账户 + 多笔交易

## 验证 US1: 流水列表

1. 登录后点底部导航"流水" tab → `/transactions`
2. 检查列表: 按时间倒序,每笔含 icon+金额+分类+账户+备注+日期
3. 若 > 50 笔,底部有"加载更多"按钮
4. 无交易时显示"暂无交易"

## 验证 US2: 筛选

1. 点击"筛选"按钮展开筛选区
2. 选"仅支出" → 列表刷新,小计仅含支出
3. 选某账户 → 列表刷新
4. 组合筛选 → AND 逻辑
5. 清空筛选 → 回到全部

## 验证 US3: 编辑

1. 点击某笔交易的编辑按钮
2. 跳转 /transaction/new?id=xxx
3. 表单预填数据
4. 修改金额 + 提交
5. 返回流水列表,数据已更新

## 验证 US4: 删除

1. 点击某笔交易删除按钮
2. `window.confirm` 弹出 "确认删除?"
3. 点"确认" → 交易消失,列表刷新
4. 点"取消" → 不删除
